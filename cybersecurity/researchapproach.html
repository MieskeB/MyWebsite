Research approach

The research approach for my cyber security minor

Main question

How can we capture critical personal data 100% securely and reliably and unlock it once with the owner's approval, with the ease of a 'Tikkie'?

Problem relevance

With this tool, we can open shops which are still closed due to COVID-19. If we can easily share that someone does not have COVID-19, we can open these shops.

Methods

What ways are most suitable to be used for authentication for the platform?

→ Community Research / Literature Study
→ Prototyping
→ Security Test

How can we securely store the data?

→ Best good and bad practices
→ Guideline Conformity Analysis
→ Security Test

How can we securely process the data?

→ Observation (how do other websites do it?)
→ Guideline Conformity Analysis
→ Security Test

How can we fulfill the reliability requirements of the stakeholders?

→ Stakeholder analysis
→ IT architecture sketching
→ Pitch

How can we assure that the platform complies to the Dutch AVG laws?

→ Literature study
→ Multi-criteria decision making
→ Guideline Conformity Analysis

How can we verify that the platform is secure?

→ Community Research / Literature Study
→ Available Product Analysis (tools that check security)
→ IT Architecture Sketching
